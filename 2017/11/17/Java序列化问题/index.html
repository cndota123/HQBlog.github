Java序列化问题

Spring Oauth2在使用数据库存储token的时候，会将生成的token对象序列化存储到数据库，
此时用于验证用户的方法中，涉及到对用户表的查询，因此必须将用户表映射到java的对象也继承Serializable,也就是序列化的能力，否则报notserializableexception。
