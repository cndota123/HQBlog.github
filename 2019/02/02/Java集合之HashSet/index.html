Java集合之HashSet
发表于2019-02-02|更新于2020-09-05|java
|字数总计:499|阅读时长:2分钟|阅读量:

HashSet的实现基于HashMap，一个HashSet实例创建的时候会对实例变量map初始化，每次add的数据作为key值添加到map中其中value值位new Object()。HashSet的遍历就是HashMap的key遍历。
