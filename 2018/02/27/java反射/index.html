java反射

类型信息

Java让我们在运行时识别对象和类的信息。主要有两种方式：一种是传统的RTTI，他假定我们在编译时已经知道了所有的类型信息；另一种是反射机制，它允许我们在运行时发现和使用类的信息。

反射的使用

  2. 任何数据类型（包括基础数据类型）都有一个静态的class属性
  3. 通过Class类的静态方法：forName（String className）
